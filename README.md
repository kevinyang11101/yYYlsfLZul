# 热门电影推荐系统 python1011

## 项目概述

本项目是一个基于机器学习的热门电影推荐系统，主要使用 Python 语言，结合 Django 框架和 MYSQL 数据库进行开发。

## 技术栈

- **后端**: Python
- **框架**: Django
- **数据库**: MYSQL

## 功能模块

### 前端功能

- **首页**: 展示系统概览。
- **热门电影**: 展示当前热门的电影列表。
- **猜你喜欢**: 根据用户历史评分推荐电影。
- **电影分类**: 展示不同类型的电影列表。
- **电影详情**: 展示电影的详细信息。
- **基于用户评分推荐**: 通过分析用户评分数据，为用户推荐合适的电影。
- **我的评分历史**: 展示用户过往的评分记录。

### 后端管理功能

- **后台登录**: 管理员登录入口。
- **热门电影列表管理**: 管理热门电影列表信息。
- **用户列表管理**: 对用户信息进行管理。
- **用户评分列表管理**: 管理用户的评分数据。
- **电影列表管理**: 对电影信息进行管理。
- **电影相似度列表管理**: 管理电影间的相似度数据。
- **电影类型管理**: 对电影类型进行管理。
- **电影评分管理**: 管理电影的整体评分。

## 系统角色

- **用户**: 可以查看电影信息，进行电影评分，查看推荐电影。
- **管理员**: 可以登录后台，进行电影和用户信息的管理。
